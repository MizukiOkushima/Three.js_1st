# Three.js_1st

### 目的
・Three.jsの基礎的学習<br>
・camera, scene, renderer, meshの基本概念<br>
・関数の使い方および具体的な引数の設定のやり方<br>

・マウスクリックでズーム＆ズームアウト<br>
・マウスカーソルの位置でカメラ移動<br>
